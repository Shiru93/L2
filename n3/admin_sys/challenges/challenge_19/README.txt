Le fichier nommé ./sandbox/default correspond au fichier de configuration par
défaut du serveur web nginx. Il contient beaucoup de commentaires introduits
par un # permettant:

  - d'introduire le fichier (cf le premier bloc)
  - d'expliquer certaines lignes de configuration
  - de fournir des exemples de blocs  possibles que vous pourriez vouloir
    copier pour les utiliser dans votre propre configuration (cf le dernier
    bloc).

Depuis votre shell, en une seule commande, créez un fichier nommé
./sandbox/sans_commentaire contenant exactement les lignes du fichier nommé
./sandbox/default ne contenant pas de commentaire.
